# App Support Site

Static personal and support website for mobile applications by Sangyoon Woo.

## Structure

- `index.html` contains the page content, app list, support information, and privacy notices.
- `assets/styles.css` contains all styling.
- `app-ads.txt` is retained for app advertising configuration.

To add another app, duplicate an app item in `index.html`, update the store links, and add a matching privacy notice.
